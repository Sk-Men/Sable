# Sable Client Changelog

## Version 0.0.1

- Added a reset button to the room name input box for dms.
  - Reset's the dm room name to the name of the other user (on both ends).
  - Same as saving a blank room name.
- Added Cosmetics tab to room and space settings.
- Added /color command that controls your name color for a room.
  - Allows setting a color if permission level is above 50 (default)
  - Added toggle in cosmetics for rooms that lowers that permission level to 0, accessible by room Administrators.
- Added /gcolor command that controls your name color for a space.
  - Allows setting a color if permission level is above 50 (default)
  - Added toggle in cosmetics for spaces that lowers that permission level to 0, accessible by space administrators.
- Name heiarchy: Room Color > Space Color > Role Color > Default
- Includes all features in Cinny v4.10.3
